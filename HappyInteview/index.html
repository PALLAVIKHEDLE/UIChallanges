• what is JS? role of js engine

    JS is a programming language that is used for converting static pages to interactive and dynamic web pages.

    JS engine is a program present in the web browser that executes js code.

  • what is Scope

    Scope determines where variables are defined and where they can be accessed.

    Types of scope

    • Block Scope
    • Global Scope
    • Function Scope

  • What is hoisting

    Hoisting is a JS behaviour where Functions and variable declaration are moved to the top of their respective scope during the compilation phase.

    JS engine will rearrange the code.

    Types of Hoisting

    • Variable Hoisting
    • Function Hoisting

    Hoisting will work only with declaration not with initialization.

  • What is JSON?

    JSON(java script object Notation) is a lightweight data interchange format. consist of key value pair.

  • Short circuit evaluation in js

    Short circuit evaluation stops the execution as soon as the result can be determined without evaluating the remaining sub-expression.

    let result =true || someFunction(), // output(true)

  • Spread operator

    The Spread operator(...) is used to expand or spread elements from an iterable(such as an array, string or object) into individual elements.

    const arr=[1,2,3]

    console.log(...arr)// output: 1,2,3

    Purpose of spread operator

    • Copying an array
    • Merging arrays
    • Passing multiple arguments to a Function

  • Rest operator

    The Rest operator(...) is used in function parameters to collect all remaining arguments into an array.

    display(1,2,3,4,5)
    function display(first, second, ...restOperator){
    console.log(first)
    console.log(second)
    console.log(restOperator)
    }
